# Mantenimiento

## Política de Mantenimiento

- Se realizarían actualizaciones según los requerimientos de los usuarios

- Para el soporte se estará disponible entre semana de la mañana a la tarde

## Changelog

- Version 0.1: Se implementa la funcionalidades del estudiante (el estudiante pude ver las asignaturas en las que está matriculado y ver sus notas)
- Versión 0.2: Se implementa la funcionalidades del profesor (el profesor puede ver la lista de usuarios que pertenecen a las asignaturas que imparte)
- Versión 1.0: Se implenta la relación entre usuarios y profesores, de manera que el profesor puede añadir notas a estos. Los estudiantes pueden solicitar un certificado de notas.

- El control de versiones de la aplicación se ha llevado a cabo mediante un repositorio de GitHub.

## Plan de Escabilidad

- En un principio no esperamos que la aplicación supere los 500 usuarios simultáneos, en caso de que la aplicación gané popularidad hemos contemplado la posibilidad de aumentar la cantidad de usuarios simultáneos que pueda haber.

- Dar un tiempo limitado si el usuario está ausente para poder cerrar la sesión de manera automática.
