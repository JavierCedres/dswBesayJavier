# Manuales

## Instrucciones Básicas

- Nuestra aplicación es un software web por lo tanto no es necesaria ninguna instalación y configuración local

## Guías Avanzadas

- Creación de certificado de notas: Antes de poder solicitarlo el estudiante debe de tener asignadas notas en cada asignatura en la que está matriculado. Luego deberá hacer click en el enlace de Solicitar Certificado. El certificado se entregará en el correo electrónico del estudiante

- Matricularse: Una vez que el usuario haya creado su cuenta, desde la página principal podrá ver un enlace de "Matricularse", una vez dentro deberá seleccionar las asignaturas que quiere cursar.

- Asignar las notas: Para realizar esta acción el usuario deberá estar registrado como profesor, luego deberá entrar en la asignatura deseada, una vez seleccionada la asignatura encontrará un enlace para ver las notas, dentro de esa página verá otro enlace que le permitirá editar todas las notas que desee.

## Solución de Problemas

!!! FAQ

    - ¿Dónde puedo ver el certificado de notas?

    El certificado de notas se manda al correo con el que te registraste en la plataforma.

    - No puedo solicitar mi certificado de notas

    Para solicitar el certificado de notas tienes que tener notas en todas las asignaturas en las que estas matriculado.
